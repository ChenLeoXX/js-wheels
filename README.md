# js-wheels
### 组件的定义
- 我们日常工作中经常用到的就是 UI 组件
- UI 组件一般是由 HTML、CSS 和 JS 组成，把 UI 组件做好了，就能更好的做网页
- UI 组件做起来更有趣，所有效果你都能用眼睛看到
### 原则
- （内部）分层原则：正交原则
- （对外）封装原则：面向接口编程
